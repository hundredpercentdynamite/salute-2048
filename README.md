## Salute-2048

### Getting Started

```shell
$ npm install
$ npm start
# open http://localhost:3000
```

You can also access the local development server from mobile

```shell
$ npm install
$ npm run start:host
# open http://$(your_computer_ip):3000, for exmaple http://192.168.0.1:3000
```
